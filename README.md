# 오메추 (오늘의 메뉴 추천)

오늘의 메뉴를 귀엽게 추천해주는 웹 애플리케이션입니다.

## 기능

- 아침/점심/저녁 메뉴 추천
- 즐겨찾기 기능
- 다크모드 지원
- 반응형 디자인

## 기술 스택

- HTML5
- CSS3
- JavaScript (Vanilla)

## 사용 방법

1. https://sortman333.github.io/todaymenuchoice/ 접속
2. "시작하기" 버튼 클릭
3. 식사 시간 선택 (아침/점심/저녁)
4. 추천된 메뉴 확인
5. 마음에 들지 않으면 "다른 음식은 어떤가요?" 버튼으로 다른 메뉴 확인
